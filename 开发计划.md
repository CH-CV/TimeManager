# TimeWise 开发计划

> 版本：v2.0  
> 日期：2026-03-26

---

## 项目概述

**目标**：开发一款以 AI 智能为核心差异化的时间管理应用

**核心价值**：
- 简洁不纠结：功能聚焦，减少选择困难
- AI 赋能：智能安排日程，记忆用户习惯
- 多平台覆盖：iOS / Android / Web / 桌面

---

## 技术栈

| 模块 | 技术选型 |
|------|----------|
| 框架 | Flutter |
| 语言 | Dart |
| 状态管理 | Riverpod |
| 本地存储 | Hive |
| 后端 | FastAPI + Python |
| AI | OpenAI |
| 路由 | go_router |
